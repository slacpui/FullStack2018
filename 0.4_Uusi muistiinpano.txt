title Uusi muistiinpano
Käyttäjä -> Selain:

note left of Selain
Käyttäjä kirjoittaa muistiinpanon
ja klikkaa Talleta -nappia.
end note

Selain -> Palvelin: POST https://fullstack-exampleapp.herokuapp.com/new_note
note right of Palvelin
Palvelin luo käyttäjän lisäämää muistiinpanoa 
vastaavan olion, joka lisätään notes -taulukkoon.
end note

Palvelin -> Selain: status 302, uudelleenohjaus /notes -kohteeseen

Selain -> Palvelin: GET https://fullstack-exampleapp.herokuapp.com/notes
note left of Palvelin
Muodostetaan HTML, johon on lisätty
tehdyt muistiinpanot data.json -tiedostosta 
ja ulkoasu style.css -tiedostosta.
end note

Selain -> Palvelin: GET https://fullstack-exampleapp.herokuapp.com/style.css
Palvelin -> Selain: status 200, sivun ulkoasun muotoilun sisältävä CSS-tiedosto
Selain -> Palvelin: GET https://fullstack-exampleapp.herokuapp.com/main.js
Palvelin -> Selain: status 200, päivittyvän toiminnallisuuden sisältävä JavaScript-tiedosto
Selain -> Palvelin: GET https://fullstack-exampleapp.herokuapp.com/data.json
Palvelin -> Selain: status 200, muistiinpanot sisältävä JSON-tiedosto.

note left of Selain
Selain näyttää palvelimen palauttaman HTML-sivun,
johon on lisätty data.json -tiedostossa olevat muistiinpanot
ja style.css -tiedoston määrittelemä ulkoasu.
